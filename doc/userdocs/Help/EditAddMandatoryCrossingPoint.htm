Add/Mandatory Crossing Point

Adds a mandatory crossing point to a leg. Unlike optional crossing points, mandatory crossing points indicate the only safe or permitted place to cross a dangerous feature, such as a road or stream.

Mandatory crossing points behave much like controls—they appear between two particular controls, with leg lines leading to the crossing point, and they have a separate line in the control description sheet to describe them. To insert a mandatory crossing point, select the leg into which the crossing point should be inserted. Chose the Edit/Add Special Item/Mandatory Crossing Point command, and click the mouse where the crossing point should be placed. Finally, chose the Item/Rotate command and rotate the crossing point to the correct orientation.

Unlike an optional crossing point, a mandatory crossing point appears only on the course you add it to. If you wish it to appear on other courses, you must add it to each course that uses it.
