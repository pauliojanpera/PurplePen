Add/First Aid Location

Adds a special item indicating the location of first aid that is not located at a control point. Click the mouse at the location on the map of the first aid.

If first aid is located at a control point, this special item should not be used; the first aid symbol should be placed in the last column of the control description instead.

By default, the first aid location special item appears on all courses. If you wish it to appear only on some courses, use the Item/Change Displayed Courses command.
