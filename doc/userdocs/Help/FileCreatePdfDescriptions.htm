File/Create PDFs/Descriptions...

The Create PDFs/Descriptions command creates a PDF with control description sheets for one or more courses.

This dialog has several options to customize the result of the created PDF.

Output section

This section shows the paper size and orientation, and the margins. Click Change Margins to change the paper size, orientation, or margins.

Courses section

This section shows which courses will have their control description sheets printed. The default setting is for all courses except "All Controls" to be printed. Click the checkboxes next to the courses you wish to print. Click All or None to quickly select or unselect all of the checkboxes at once. If you are printing a course with variations, click on the course name (not just the checkbox), select Choose Variations... to select how to print the variations.

Copies section

This section determines how many copies of each description sheet will be printed. There are three options:

  • one copy of each course. With this option, one description will be printed for each course, with different courses being placed on the same sheet of paper. The program will attempt to arrange the control descriptions so that as few sheets of paper as possible will be used.
  • one page of each course. With the option, one page of descriptions will be printed for each course. As many copies of the same course's description will be placed on each page as will fit.
  • multiple pages of each course. With this option, you can chose how many descriptions to print for each course. The program will print as many pages as needed of each description to reach the requested number. The program will not mix different descriptions on the same page.

If you choose "multiple pages of each course", enter the number of copies of each description (not the number of pages) in the text box below. For example, if you enter 50 descriptions per course, and the program can fit 10 descriptions on each page, 5 pages of 10 descriptions each will be printed for that course. If another course can only fit 6 description on each page, then 9 pages of 6 descriptions each will be printed for that course.

Appearance section

The section determines how the descriptions appear.

Line size: The height of each line in the description sheet. Since the boxes in a description sheet are square, this also determines the width of boxes. The IOF recommends a size between 5mm and 7mm, but any size between 3 and 10mm may be specified.

Description type: The type of description sheet to use. The possible options are:

  • Symbols: the standard IOF symbolic description sheet, with 8 columns and symbols used to describe controls.
  • Text: instead of symbols, an English language textual description of the control is used. The textual description fills the area normally occupied by the last 6 columns of the description. The textual description is automatically generated.
  • Symbols and Text: The IOF symbols, with text equivalents to the right. The textual description for each control is automatically generated.
  • Course Default: Whichever of Symbols, Text, and Symbols and Text is specified in the course properties for the course.

Other buttons

Create PDF: This button creates the PDF file. The dialog box closes, and you are prompted for where to save the PDF file.

Preview: Opens a print preview dialog box that shows what will be in the created PDF.

Cancel: Closes the dialog box without doing anything.
