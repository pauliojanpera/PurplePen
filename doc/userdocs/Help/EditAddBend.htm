Item/Add Bend

Toolbar Button:

Keyboard Shortcut: Ctrl+B

Note: This command only is active when a leg, out-of-bounds area, dangerous area, or uncrossable boundary is selected.

Adds a bend into a leg, so that the leg can bend around an uncrossable obstacle. Also, adds a corner into a special item such as a dangerous area or uncrossable boundary. After selecting the Add Bend command, click the left mouse button where you would like the bend or corner to be placed.

Once a bend is added, it can be repositioned by dragging it with the mouse while the leg or special item is selected.
