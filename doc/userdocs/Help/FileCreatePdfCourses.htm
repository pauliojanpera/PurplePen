File/Create PDFs/Courses...

Creates PDF file(s) for later viewing or printing. By default, the entire map will be placed in the PDF. To chose the area that will be printed, the paper size, and the paper orientation, use the File/Set Print Area command.

Using this command is preferable to using the Print Courses command and a PDF printer driver for several reasons:

  • The resulting PDF will typically be smaller in size.
  • The resulting PDF will use vector graphics for higher quality printing.
  • The resulting PDF will use (if the default is retained), the Cyan-Magenta-Yellow-Black color model.
  • There are less likely to be problems resulting from printer driver bugs.
  • If the base map file is a PDF file, the PDF from that file will be copied directly into the output, yielding as high a quality as the original PDF.
  • If desired, each course can be placed in its own PDF file.

In some cases, using this command and printing the resulting PDFs may produce better output than directly using the Print Courses command, depending on your specific printer.

 

This dialog has several options to customize how PDF files are created.

Courses section

This section shows which courses will be printed. The default setting is for all courses except "All Controls" to be printed. Click the checkboxes next to the courses you wish to print. Click All or None to quickly select or unselect all of the checkboxes at once. If you are printing a course with variations, click on the course name (not just the checkbox), select Choose Variations... to select how to print the variations.

Files sections

Name prefix: If something is typed in here, it is prefixed onto the file name of each PDF file. For example, if you type "Final" in this box, then the result files would be "Final-Course 1.pdf", "Final-Course 2.pdf", etc.

Files: Indicates how multiple maps are packaged into PDF files. The choices are:

  • One for all courses. A single PDF file is created with all courses in it, one page per course. The name of the PDF is the name of the event.
  • One per course. Each course is placed in its own PDF file. The name of the PDF is the name of the course. If a course has a map exchange, the PDF file will contains multiple pages, one for each part.
  • One per course part. Each course is placed in its own PDF file .The name of the PDF is the name of the course. If a course has a map exchange, each part of the course is placed in its own PDF file, with "-1", "-2", etc added to the file name.

Appearance section

Several things are set in this area:

Print area too large: This setting determines what happens if the print area is too large to fit on a single sheet of paper. Crop to a single page (the default) will pick the portion of the map that best covers that course. Print on multiple pages will print multiple pages of maps to cover the entire print area. Note: if the base map file is a PDF map, you cannot change this option.

Print Map Exchanges on Same Map: This setting only appears if a map exchange is present in the event. If this is checked, all parts of the course are printed together, overlapping. If unchecked, a separate map is printed for each part of the course. Typically this should be left unchecked unless you want to see all parts together for some reason.

 Color Model: Determines how the colors on the map are stored inside the PDF file:

  • RGB: Translates the Cyan-Magenta-Yellow-Black colors in the map to Red-Green-Blue colors using a simple formula. The colors on the printer will typically not match the intended colors exactly. This is the method used by versions of Purple Pen prior to 2.0.
  • CMYK: The default. Outputs the Cyan-Magenta-Yellow-Black colors to the PDF file. This generally produces the most accurate rendition of the intended colors.

Folder section

This section determines in which folder (directory) the course files are created. There are three choices:

  • Same folder as Purple Pen file. The files are placed in the same folder as the Purple Pen (.ppen) file for this event. This is the default choice.
  • Same folder as map file. The files are placed in the same folder as the map file for this event.
  • Other folder. Any other folder. If you chose this, a "Select Folder..." button appears which you can click to select any folder.

Other buttons

Create: This button creates the PDFs. If this would cause files to be overwritten, a list of those files is displayed. The dialog box closes.

Cancel: Closes the dialog box without doing anything.
