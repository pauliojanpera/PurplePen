Printing

Once your courses are completely done, you will want to print out the final materials for your event.

There are a few different ways to print your course maps. If you are picky about output, you may wish to experiment to see which method produces the best output with your printing equipment; printers, programs, and printer drivers often vary in subtle ways that are hard to predict.

  • Directly print courses using the Print Courses command. This will directly print out your map with the courses drawn on top. In earlier versions, OCAD printing was recommended over direct printing from Purple Pen, but in version 2.0 and higher, Purple Pen prints as accurately as OCAD. Note that the "Color Model" and "Rasterize Before Printing" options may affect your output quality also.
  • Use OCAD or OpenOrienteering Mapper to print out your courses. This can be useful if you wish to use OCAD/OOM for further editing. First, export the courses to OCAD/OOM using the Create OCAD Files command. This creates one OCAD file for each course. Then, load each file into OCAD/OOM and print it. You must have the original map file as well as the exported OCAD/OOM file.
  • Create PDF files to print later or take to a printing shop. First, create PDF files using the Create PDFs command. You can then print the resulting PDF files on any printer or given them to a printer shop. When printing. be sure to print the PDF without any scaling applied (do not scale to the printable area).

You can print out the control descriptions for each course using the Print Descriptions command.

If you are using pin punching, print out master punch cards for use in control card checking using the Print Punch Cards command.

Finally, you may wish to print out a Course Summary and Control Cross-reference report for reference during the event.
