View/All Controls

Turns on or off the display of all controls in the control collection in a course view. If this view is turned on, then controls that are not in the current course are displayed in light purple alongside the current course.

Note: If you wish to simply see all the controls, it is easier to just click the "All Controls" tab at the top of the screen. The View/All Controls command is useful only if you wish to see all the controls at the same time you are viewing a course.
