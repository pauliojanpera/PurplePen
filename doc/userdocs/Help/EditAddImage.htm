Add/Image

Adds a special item that is a bitmap image to the map. The image can be a BMP, JPEG, PNG, GIF, or TIFF file.

First, select an image file using the standard file picker.

After selecting the image file, draw the location of the image on the map with the mouse as follows:

  • Press and hold the left mouse button at one corner of the image.
  • Move the mouse to the diagonally opposite corner, and release the mouse button.

By default, the image appears on all courses. If you wish it to appear only on some courses, use the Item/Change Displayed Courses command.
