Add/Forbidden Route Marking

Adds a special item indicating a forbidden route. Click the mouse at the location on the map of the forbidden route marking.

If a route on the map is forbidden to competitors, the forbidden route marking should be placed as many times as necessary all along the forbidden route.

By default, the forbidden route special item appears on all courses. If you wish it to appear only on some courses, use the Item/Change Displayed Courses command.
