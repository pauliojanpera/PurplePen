Add/Registration Mark

Adds a special item that is a small purple cross, for the purpose of ensuring that overprinting is occurring correctly. Click the mouse at the location on the map of the registration mark.

Registration marks are usually placed on top of corresponding registration marks on the underlying map. If you are overprinting courses onto existing maps, registration marks are useful to ensure that overprinting occurs correctly. The current version of Purple Pen does not directly support overprinting, however.

By default, the registration mark special item appears on all courses. If you wish it to appear only on some courses, use the Item/Change Displayed Courses command.
