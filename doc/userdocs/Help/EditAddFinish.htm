Add/Finish

Toolbar Button:

The Add Finish command adds a finish to the end of the currently displayed course. If the course already has a finish, it is replaced with the new finish.

When this command is selected, any existing finish controls in the control collection that are not already in the current course are shown in light purple on the current course map.

  • To add an existing finish to the current course, click on the light purple existing finish.
  • To add a new finish to the current course, click elsewhere on the map.

When a new finish is created, it is added to all courses that do not already have a finish control.
