Reports/Event Audit

The Event Audit report checks the event for a number of different kinds of potential problems, and reports any that are found. The Event Audit consists of a number of sections. If no problems are found for a given section, that section is omitted. The sections are as follows:

Missing Items in Courses

  • Reports any course without a start triangle.
  • Reports any normal course without a finish circle.
  • Reports any normal course without a climb amount.
  • If some courses have competitor loads defined, reports any course without a competitor load.

Close Together Controls

  • Reports any pair of controls closer than 100m apart.

Unused Controls

  • Reports all controls, starts, finishes, and mandatory crossing points that are in the All Controls collection, but not used in any course.

Missing Description Boxes

  • Reports any control with no main symbol in column D.
  • Reports any controls using the "crossing", "junction", or "between" symbols that does not have a symbol in both columns D and E.

Missing Punch Patterns

  • Reports all controls without a punch pattern defined. If none of the controls have punch patterns, then this section is omitted.

Missing Points

  • Reports all controls in a score course that don't have points assigned to them. If none of the controls in the course have points assigned, then this section is omitted.

 

To print the report, click the Print button. To view a preview of the printed report, or to adjust margins, text size, headers and footers, or paper orientation, click the Print Preview button.

To copy all or part of the report to the clipboard, select the portion of the report you wish to copy. Then click the right mouse button on the selected text, and select Copy from the menu.
