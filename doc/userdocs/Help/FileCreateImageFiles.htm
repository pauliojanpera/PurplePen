File/Create Image Files...

This command creates an image (bitmap) file for each course that you specify. This can be useful for displaying a course on a web site, or for further editing in a paint program.

For each course chosen, an image file with the name of that course will be created, using the print area defined for that course.

 

This dialog has several options to customize the result.

Courses section

This section shows which courses will have image files created. The default setting is for all courses except "All Controls" to be printed. Click the checkboxes next to the courses you wish to print. Click All or None to quickly select or unselect all of the checkboxes at once. If you are exporting a course with variations, click on the course name (not just the checkbox), select Choose Variations... to select how to export the variations.

For each course, an image file with the name of the course is created.

Output section

File format: This shows which file format will be used; either PNG, JPEG, or GIF.

DPI: This number of dots per inch in the bitmap. The higher the number, the more detail is shown, and the larger the image file will be.

Color Model: Determines how the colors on the map are converted into colors on the printer:

  • RGB: Translates the Cyan-Magenta-Yellow-Black colors in the map to Red-Green-Blue colors using a simple formula. The colors in the image will typically not match the intended colors exactly.
  • CMYK: The default. Keeps the Cyan-Magenta-Yellow-Black colors as true as possible, using a standard color profile of CMYK colors. This generally produces the most accurate rendition of the intended colors.

World file: This is enabled if the base map file is an OCAD or OOM file with real world coordinates defined. Set this to Yes to create a .PGW, .JGW, or .GFW world file that defines the relationship between the image file and the real world coordinate system.

File name prefix: If something is typed in here, it is prefixed onto the file name of each output file. For example, if you type "Final" in this box, then the result files would be "Final-White.png", "Final-Yellow.png", etc.

Folder section

This section determines in which folder (directory) the course files are created. There are three choices:

  • Same folder as Purple Pen file. The files are placed in the same folder as the Purple Pen (.ppen) file for this event. This is the default choice.
  • Same folder as map file. The files are placed in the same folder as the map (.ocd/.xmap/.omap) file for this event.
  • Other folder. Any other folder. If you chose this, a "Select Folder..." button appears which you can click to select any folder.

Other buttons

Create: This button Creates the image files. The dialog box closes.

Cancel: Closes the dialog box without creating anything.
