File / Program Language

This command allows you to set the language that Purple Pen will display it's user interface in. The list shows all the available languages that Purple Pen has been translated to. Select the language that you wish to use. Your choice will be remembered on your computer for running Purple Pen subsequent times.

Note: The languages used for textual control descriptions is set separately. To set the language that textual control descriptions will be printed in, use the Event / Customize Description Text command.
