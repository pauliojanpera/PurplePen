Add/Text

Adds a special item that is text. The text can be any size. The color and font of the text is customizable.

Part (or all) of the text can be "special text" pieces, which are replaced with appropriate text differently for each course. This can be used, for example, to place the name of each course on the map. The special texts are:

$(EventTitle) The main title of the entire event.
$(CourseName) The name of the course
$(CoursePart) The part number of a course with a map exchange.
$(CourseLength) The length of the course, in km.
$(CourseClimb) The climb of the course. Blank if not climb was specified.
$(ClassList) The secondary title or class list for the course, if any.
$(PrintScale) The printing scale for this course.
$(RelayTeam) The relay team number. This special text only takes effect when selecting "Each Relay Leg Separately" when choosing variations to print.
$(RelayLeg) The relay leg number. This special text only takes effect when selecting "Each Relay Leg Separately" when choosing variations to print.
$(FileName) The file name of the Purple Pen file.
$(MapFileName) The file name of the map file used.

To add text, select the Add Special Item/Text command.

First, you enter the text to display in a dialog:

Type the text you wish to display. The "Insert Special Text" button can be used to easily insert one of the special texts from the table above.

You can customize the appearance of the text on the map as follows:

    Font: Select the name of the font you wish to use. You can see a preview of the text in the "Sample" box.

    Bold: Check the box to boldface the text.

    Italic: Check the box to make the text italic.

    Color: Select the color of the text. You can select one of the predefined colors from the list. If you select "Purple", the color of the text will match the purple color used for the rest of the course (which can be set through the Customize Appearance command.)

    Change Color: If you wish to select a color other than the predefined colors in the list, click this button and choose the amount of Cyan, Magenta, Yellow, and Black in the color that you want.

    Size: If Automatic is not checked, determines the size of the text. The size indicates the height, in mm, of a capital letter or digit.

    Automatic: If checked, indicates that the text size is set to fill the rectangle.

 

After clicking "OK", then press and hold the left mouse button. Drag the mouse to size the text box as desired. When the text is the correct size, release the left mouse button.

 

Note: once the text is placed, use the Item/Change Text... menu item to change the text, the font, or the color.
