Edit/Add Corner

Note: This command only appears when a out-of-bounds area, dangerous area, or uncrossable boundary is selected.

Adds a corner into a special item such as a dangerous area or uncrossable boundary. After selecting the Add Corner command, click the left mouse button where you would like the new corner to be placed.

Once a corner is added to a special item, it can be repositioned by dragging it with the mouse while the special item is selected.
