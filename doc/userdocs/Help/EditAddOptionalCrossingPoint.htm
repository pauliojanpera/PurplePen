Item/Add Special Item/Optional Crossing Point

Adds an optional crossing point. A crossing point indicates a safe or allowed place to cross a dangerous or forbidden feature, such as a road or river. An optional crossing point simply adds the crossing point symbol to the map, but does not change the leg marking or control descriptions to indicate that the crossing point is required. If the crossing point is the only safe option for a particular leg, the Item/Add Special Item/Mandatory Crossing Point command should be used instead.

Chose the Edit/Add Special Item/Optional Crossing Point command, and click the mouse on the location of the crossing point. Then use the Item/Rotate command to rotate the crossing point to the desired angle.

By default (and unlike a mandatory crossing point), an optional crossing point appears on all courses. If you wish it to appear only on some courses, use the Item/Change Displayed Courses command.
