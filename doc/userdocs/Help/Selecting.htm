Selecting objects

In order to perform many operations in Purple Pen, you must first select the object that you wish to work on. There are three general types of objects you can select.

Selecting a control, start, or finish

There are two ways to select a control, start, or finish.

  • Clicking on the control circle in the map display.
  • Clicking on the control line in the control description pane.

When a control, start, or finish is selected, the control circle is displayed in red, and the corresponding control description line is highlighted in yellow. The selected object description pane shows the control code, location, and which courses use that control.

Selecting a leg

To select a leg, click on the line representing the leg in the map display.

When a leg is selected, it is displayed in red. The selected object description pane shows the length of the leg, and which courses use that leg.

Selecting a special item

Click in the map on a special item such as a first aid location, out-of-bounds area, or control description block to select it.

When a special object is selected, it is displayed in red. The selected object description pane shows which courses display that special item.

Moving selected items

To move an item, it must first be selected. Once it is selected, most items can be moved by holding the mouse down on them and dragging them to a new position. Control numbers or codes can be repositioned by moving them also. Once a control number is moved, it will always stay in that location. To make a control number be automatically positioned again, drag it onto the control circle.
