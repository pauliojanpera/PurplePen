Selected Object Pane

The selected object pane is located in the bottom left corner, immediately below the control description pane. The size of the selected object pane can be adjusted by dragging the top border of the pane with the mouse.

The selected object pane displays detailed information about the currently selected object. The exact details displayed depend on what kind of object is selected, as follows:

Control, Start, or Finish

  • The control code.
  • The exact paper coordinates.
  • Which courses use this control.
  • The textual version of the description for this control. This description appears if the textual version of the control descriptions are printed.

Leg

  • The control codes of both ends of the leg.
  • The length of the leg.
  • Which courses use this leg.
  • Whether the leg includes any flagged portions.

Special Item

  • The type of special item.
  • The exact paper coordinates.
  • Which courses display this special item.

Nothing Selected (any course)

  • The name of the course.
  • The length of the course.

Nothing Selected (all controls)

  • The number of controls currently is use by one or more courses.
  • The number controls currently not being used by any course.
