Moving around the map

Panning the map

There are three ways to pan around the map to see parts that are off the edge of the display window.

  • Use the scroll bars on the right and bottom edges of the map windows to move the map around.
  • Press and hold either the left or right mouse button, and drag the map around.
  • Press the arrow keys on the keyboard.

Zooming in and out

There are three ways to zoom the map in and out. The current zoom level is always displayed at the bottom right of the main window, just to the left of the zoom slider.

  • Use the View/Zoom menu item to select a zoom level.
  • Drag the zoom slider, located at the bottom right of the main window.
  • Use the mouse wheel to zoom in and out. When using the mouse wheel, the map will zoom around the point where the mouse pointer is.

Shortcuts

There are two shortcut commands that can be useful:

  • The View/Entire Map command positions and zooms the main window to view the entire map.
  • The View/Entire Course command positions and zooms the main window to view all the controls on the current course.

Tip: Experienced users find that using the mouse wheel to zoom is often the easiest way to move long distances around the map. Zoom out until the area you wish to view is visible, then move the mouse to that location and zoom in.
