Edit/Delete

Toolbar Button:

Keyboard Shortcut: Del

Deletes the currently selected control (or special object).

If the "All controls" map is currently displayed, the selected control is deleted from the controls collection and all courses that it is used in.

If a course map is currently displayed, the control is deleted only from the current course. If the same control is not used in any other course, you will be asked if you wish to delete the control from the control collection as well.
