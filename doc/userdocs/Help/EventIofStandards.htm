Event/IOF Standards

The IOF Standards menu controls the IOF standards used for the event. There are two different choices that can be made for the IOF standard—the control description standard and the mapping standard.

There are two choices for the control description standard:

  • 2004 Description Standard. The IOF control description standard published in 2004.
  • 2018/2024 Description Standard. The IOF control description standard published in late 2017: download here. This standard has new symbols for trench and flower bed/out of bounds area. The bend symbol moves to a different column. Select this also for the IOF controls description published in late 2023: download here. The new railway and map flip symbols are added.

There are two choices for the maps standard. The choice of standard slightly affects the standard sizes for circles and other items overprinted in purple on the map, and the appearance of out of bounds areas.

  • ISOM 2000. The International Standard for Orienteering Maps published in 2000: download here.
  • ISOM 2017. The International Standard for Orienteering Maps published in 2017: download here.
