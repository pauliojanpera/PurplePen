Event/Change Codes...

Displays a dialog that allows changing any or all of the control codes in the event.

To change a control code, click in the "New Code" column opposite the code you wish to change, and type in the new code. Changed control codes are displayed in red. A control code must consist of two or three letters or digits, such as "48", "502", "GO", or "W12".

If you wish to change all existing controls to be numbered in sequence, you can use the Controls/Automatic Numbering command to renumber all existing controls in order.

Selecting the Undo command after completing the Change Codes... command will undo all the changes made.
