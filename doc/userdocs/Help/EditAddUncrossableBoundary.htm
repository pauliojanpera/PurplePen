Add/Uncrossable Boundary

Draws a line indicating a boundary that is forbidden for competitors to cross.

Draw the line on the map with the mouse as follows:

  • Press and hold the left mouse button at the location where the line should begin.
  • Move the mouse to where the first corner should appear, and release the mouse button.
  • Press and hold the left mouse button again to draw the next line segment.
  • Move the mouse to where the next corner should appear, and release the mouse button.
  • After releasing the mouse button at the end of the line, click the mouse once to finish drawing.

By default, the uncrossable boundary special item appears on all courses. If you wish it to appear only on some courses, use the Item/Change Displayed Courses command.
